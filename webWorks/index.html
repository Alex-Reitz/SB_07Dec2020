What is HTTP?

hyper text transfer protocol

What is a URL?

Uniform Resource Locator

What is DNS?

Domain Name System, URL to IP Address

What is a query string?

Parameter in a URL that can be used as a argument in GET requests. Assigns values to specified parameters. Allows us to pass key-value pairs into the URL.

What are two HTTP verbs and how are they different?

Get - for no side effects, search bar, click link, some form submissions
Post - has side effects, update a website, comment, add photo, post, credit card information

What is an HTTP request?

Message that is sent from a client to a server

What is an HTTP response?

Made by a server to a client.

What is an HTTP header? Give a couple examples of request and response headers you have seen.
HTTP headers allow the client and server to pass additional information with a request or a response.
Request headers: Host, User-agent, accept, cookie, cache-control Response Headers: Content-type, Last-modified, Set-cookie, Cache-control

What are the processes that happen when you type “http://somesite.com/some/page.html” into a browser?
Browser resolves name into an IP address, Browser makes a req to that IP address, server sends a response, Browser contructs DOM, finds all resources needed including HTML, CSS, JS, Browser makes separate HTTP reqs for each resource and takes response from server for each
