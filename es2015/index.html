var is function scoped whereas let is block scoped. Can reassign or redeclare with var but cannot redeclare using let

var is function scoped and can be reassigned or redeclared wheras const is block scoped and cannot be reassigned

let variables can be updated but not redeclared, const variables can not be updated or redeclared
